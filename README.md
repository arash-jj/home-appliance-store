## Installation 💻
```bash
# Clone repository
git clone https://github.com/arash-jj/home-appliance-store.git
```
then run the development server:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
